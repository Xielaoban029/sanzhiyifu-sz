# 河北三支一扶 · 时政备考系统

📖 **在线访问**：https://xielaoban029.github.io/sanzhiyifu-sz/

## 功能

- 📖 **学习** — 知识点卡片浏览，重难点高亮标注
- ✍️ **刷题** — 单选/多选/判断，即时反馈+解析
- 🃏 **闪卡** — 间隔重复记忆，四级评分
- 📊 **统计** — 学习进度、薄弱分析、15天冲刺计划

## 自动化

- 每天 UTC 23:00（北京时间 07:00）通过 GitHub Actions 自动抓取最新时政
- 自动生成知识点和练习题
- 多源冗余 + 缓存池保底，保证每日更新
